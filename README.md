# 🎬 YouTube Clone - Frontend

This is the frontend of a full-stack YouTube Clone built using React (Vite). It allows users to register, login, upload videos, and view videos in a modern UI.

---

## 🚀 Live Features

- 🔐 User Authentication (Login / Register)
- 📺 View all uploaded videos
- ⬆️ Upload new videos (Protected)
- 🔎 Search videos by title/channel
- 🧑 Channel-based video system
- 🎨 Responsive UI using Tailwind CSS

---

## 🛠️ Tech Stack

- React (Vite)
- React Router DOM
- Axios
- Tailwind CSS
- Context API (Auth Management)
